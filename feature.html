Play Hub App

Play hub app contains mind games like TicTacToe (Single & Multi Player), Othello or Reversi (Single & Multiple Player) & Mine Sweeper with different levels.

Features:

  • Beautiful User Interface

    Easy to use and pleasurable ui design.

  • Tic Tac Toe

    You can choose different game type : Single player or Multi Player.There are also different board size available like 3x3,4x4,5x5,6x6.

  • Reversi

    User can choose different game type : Single player or Multi Player.

  • Mine Sweeper

    - Customize difficulty (rows, columns, mines) or use preset difficulties (easy, medium, hard)
    - Long tap to add Flag
    - First sweep is always safe
    - Flag placement is restricted, users can't place any more flags than necessary
    - Timer tracks game time, and high score tracker works for preset difficulties

  • Music and sound

    Music and sound options for enhance the user experience
